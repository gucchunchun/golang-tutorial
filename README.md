## 課題2: 変数の宣言と初期化（var, :=）を使い分ける

### 学び
- var
  - 初期値の宣言を行わない場合に適切
  - 初期値を宣言しない場合はzero-valueが設定される
    bool: false
    int / float: 0
    string: ""
  - ```var()```とすることで複数変数を宣言できる
- :=
  - 初期値の宣言を行う場合に有効。型指定しなくても良い。
